form 관련태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그
form 태그 내의 input 태그들을 통해 사용자가 입력한 정보들을 서버로 넘기는 역할
*action 속성: 폼태그 내부에 입력된 값들을 전송받을 서버를 입력 (값을 어디로 보낼것인지)

 

submit 버튼을 클릭했을때 action에 지정한 위치로 method에 지정한 방식으로 input 태그에 입력한 value 값을 전달한다.
